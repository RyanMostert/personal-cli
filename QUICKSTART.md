# Personal-CLI Quickstart

Welcome to Personal-CLI, the AI-driven terminal workspace.

## Basic Setup

If you haven't already, link an LLM provider:
1. Type `/provider` to open the Provider Manager.
2. Select your preferred provider (e.g., Anthropic, OpenAI, OpenCode).
3. Paste your API Key.

## Essential Commands
- `/help` - Show all commands.
- `/model` - Switch between different LLMs (e.g. Claude 3.5 Sonnet to GPT-4o).
- `/history` - View and load past conversations.
- `/clear` - Clear the current conversation buffer.

## Workflow Modes
Press **Tab** to cycle between modes:
- **ask**: Standard conversation, the AI answers your questions.
- **plan**: The AI creates an execution plan before taking action.
- **build**: The AI autonomously executes tasks and file edits.

## Working with Files
- **Attach Files**: Drag and drop a file into the terminal, or type `@filename`.
- **Paste Images**: Copy an image and press `Ctrl+V` to attach it for vision models.
- **Side Panel**: Use `Ctrl+X` then `L` to toggle the side panel when editing files.
